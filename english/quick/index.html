    Getting Started

    Your Project's Dashboard

    The Project Dashboard contains all the links to your DAaaS GEA resources, we recommend bookmarking the URL.

    Find your Dashboard here: GAE Central Dashboard

    Learn more about Dashboards

    How to launch your VM

    Full Help: Link to VM Help Page

    Quick Help:

    1. From your Project Dashboard, click on your VMs name to open the Overview page
    2. In the VM Overview page, select Start and wait until the VM is running
    3. Select either Browser Connect or Connect > Bastion
    4. Enter your VM username and password
    DAaaS GEA ArcGIS Enterprise Portal

    Full Help: Link to Help ArcGIS Enterprise Portal Page

    Quick Help:

    1. Access via this URL: DAaaS GEA Enterprise Portal
    2. Click the StatCan Azure Cloud Login
    3. If this is your first time accessing it, you will have to do some registration steps, then contact DAaaS GEA to have your account configured

    How to contact DAaaS GAE

    Full Help: Link to Contact Help Page

    Quick Help:

    For support, or requests please submit a ticket via Jira:

    1. Open Statistics Canada Jira software: DAaaS Geospatial Platform (DASGEO)
    2. From the Menu Bar, select the button “Create”:
    3. In the Jira Create Issue Window, Select the Issue Type = “Client Support”:
    4. In the Jira Create Issue window, enter the information related to your request
