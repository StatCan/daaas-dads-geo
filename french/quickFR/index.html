    Trouvez votre tableau de bord ici : Tableau de bord central de l’EAG

    En savoir davantage sur les tableaux de bord

    Comment lancer votre machine virtuelle (MV):

    Aide complète: Lien vers la page d’aide relative aux MV

    Aide rapide:

    1. Dans le tableau de bord de votre projet, cliquez sur le nom de vos MV pour ouvrir la page Vue d’ensemble.
    2. À partir de la page Vue d’ensemble de la MV, cliquez sur Démarrer et attendez que la MV soit en cours d’exécution.
    3. Sélectionnez Connexion par navigateur ou Connecter > Bastion.
    4. Saisissez le nom d’utilisateur et le mot de passe de votre MV.
    Portail ArcGIS Enterprise – EAG-ADS:

    Aide complète: Lien vers la page d’aide du portail ArcGIS Enterprise

    Aide rapide:

    1. Accès avec cette URL: Portail d’entreprise EAG-ADS
    2. Cliquez sur StatCan Azure Cloud Login (connexion au nuage Azure de StatCan).
    3. Si c’est la première fois que vous y accédez, vous devrez effectuer quelques étapes d’enregistrement, pour ensuite communiquer avec l’équipe de l’EAG-ADS pour que votre compte soit configuré.

    Comment communiquer avec l’équipe de l’EAG-ADS:

    Aide complète : Lien vers la page d’aide Contact

    Aide rapide:

    Pour obtenir du soutien ou présenter des demandes, veuillez soumettre un billet JIRA :

    1. Ouvrez le logiciel JIRA de Statistique Canada : Plateforme géospatiale ADS (DASGEO)
    2. Dans la barre de menu, cliquez sur le bouton Créer :
    3. Dans la fenêtre Créer un ticket JIRA, sélectionnez le type de demande Client Support:
    4. Dans la fenêtre Créer un ticket JIRA, saisissez les renseignements relatifs à votre demande
