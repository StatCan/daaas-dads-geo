    Anaconda & Python

    Anacoda

    Warning

    Pour l’instant, Anaconda Navigator est uniquement disponible en anglais.

    Anaconda est une distribution de source ouverte des langages Python et R qui simplifie la gestion et le déploiement de paquets.

    Anaconda exploite le système de gestion de paquets, conda, qui analyse l’environnement actuel avant l’exécution et l’installation pour éviter de perturber d’autres cadres et paquets. Anaconda Navigator permet aux utilisateurs de lancer des applications et de gérer les paquets, les environnements et les canaux Conda sans utiliser la ligne de commande.

    Pour créer un nouvel environnement à l’aide de l’interface utilisateur graphique:

    Pour créer un nouvel environnement en utilisant une fenêtre CMD ou Python :

    Créer un nouvel environnement (remplacer <__>) : 

    conda create --name <env_name> <package>

    Activer votre environnement (remplacer <__>): 

    conda activate <env_name>

    Installation des paquets: La plateforme d’ADS utilise Artifactory pour la gestion des paquets et des bibliothèques :``` https://jfrog.aaw.cloud.statcan.ca/artifactory/conda-forge-remote 

    À utiliser:
```python
conda install -c https://jfrog.aaw.cloud.statcan.ca/artifactory/conda-forge-remote/ [package]

    Pour une version de paquet particulière : 

    conda install -c https://jfrog.aaw.cloud.statcan.ca/artifactory/conda-forge-remote/ [package=X.X...]

    Exemple : 

    conda create --name geo
conda activate geo
conda install -c https://jfrog.aaw.cloud.statcan.ca/artifactory/conda-forge-remote/ geopandas
conda install -c https://jfrog.aaw.cloud.statcan.ca/artifactory/conda-forge-remote/ matplotlib=3.5.2
